# Jialong Fan Academic Homepage

This is a static academic homepage prepared for PhD applications.

## Files

- `index.html`: homepage content
- `styles.css`: page styling
- `assets/research-map.svg`: simple research-theme visual
- `files/Jialong_Fan_CV.pdf`: CV linked from the homepage

## Quick Deployment With GitHub Pages

1. Create a public GitHub repository, for example `jialongfan.github.io`.
2. Upload all files inside this `homepage` folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set source to `Deploy from a branch`, branch `main`, folder `/root`.
5. The homepage URL will usually become `https://jialongfan.github.io/`.

If the repository is named differently, the URL will usually be:

```text
https://YOUR_GITHUB_USERNAME.github.io/REPOSITORY_NAME/
```

## Suggested URL for Forms

Use the deployed GitHub Pages URL as the homepage URL. A local file path such as
`C:\...` will not work in an application form.
